    2. It stands for Hyper Text MarkUp Language:

    • A markup language is a computer language that defines the structure and presentation of raw text.
    • In HTML, the computer can interpret raw text that is wrapped in HTML elements.

  2. Anatomy of HTML:

    3. HTML is composed of elements. These elements structure the webpage and define its content.
    Html element is consits:

    • Opening tag: Greater than sign, tag name and less than sign.
    • Content: Every raw text, images, video or link inside the opening and ending tag is known as content of that element.
    • Closing tag: Greater than sign, oblig or slash sign, tag name and less than sign.
    • HTML tag: the element name, surrounded by an opening (<) and closing (>) angle bracket.
    • HTML element: a unit of content in an HTML document formed by HTML tags and the text or media it contains.

    Html element's name is the tag name that occupied by opening and closing tag. And the whole value opening tag, closing tag and content are called together as 'HTML elemnts'.
    Html element is looks like:

    <tagname> ....Content..... </tagname>

    For example:
    <h1>Krishna</h1&>

  3. Body:

    4. One of the most important HTML element we used to create web page is 'body'. Only content inside the opening and closing body tags can be displayed to the screen.
    It looks like:

    <body>
    ....Content.....
    </body>

    Once your webpage has body, you can add multiple content like; text, images, videos, link and many more.
    We use it like:

    <body>
    <p>What's up, doc? </p>
    </body>
